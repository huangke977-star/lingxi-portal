# HLOVET 账号外观与个人中心设计

## 目标

- 主题、透明度、磨砂程度、磨砂颜色、磨砂透明度和头像保存到账号。
- 用户更换设备登录后可以恢复自己的外观设置。
- 个人中心不再展示冗余账号事实表，角色信息直接放入账号卡。
- 顶部导航角色标识使用图标，悬浮提示只显示角色名称。

## 外观设置

- 推荐主题和自定义配色互斥。
- 卡片透明度、磨砂程度、磨砂颜色、磨砂透明度与主题同级保存。
- 磨砂程度允许为 `0`，此时背景虚化为 `0px`。
- 磨砂透明度控制背景洗色强度，允许为 `0`。
- 自定义配色包含强调色、卡片底色、主文字和辅助文字。

## 头像

- 登录用户可以上传自己的头像。
- 支持 JPEG、PNG、WebP。
- 单张头像最大 2 MB。
- 头像文件保存在 `avatar_uploads` Docker 命名卷中。
- 更换头像后会删除旧头像磁盘文件。

## 等级说明

- 个人中心展示练气、筑基、金丹、元婴、化神、炼虚、合体、大乘。
- 管理员和超级管理员不展示在等级说明列表中。
- 练气标注为登录自动获取。
- 其他等级标注为未开放，不展示获取说明。
