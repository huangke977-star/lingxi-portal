# HLOVET 玻璃卡片门户设计

日期：2026-07-10

## 目标

将站点品牌统一改为 `HLOVET`，把上一版工作台侧边栏风格调整为浅色、卡片式、带轻微悬浮和磨砂背景的个人门户体验，并从门户入口中移除后台管理入口。

## 视觉方向

视觉主题：浅色、轻盈、玻璃质感、柔和阴影、卡片悬浮；整体像个人门户和工具入口，而不是企业工作台。

内容规划：

- 顶部导航展示 `HLOVET`、导航、工具、工作台和右上角账号区。
- 未登录时右上角显示登录入口。
- 登录后右上角显示等级徽章和用户头像。
- 鼠标悬浮等级徽章时展示具体用户等级。
- 鼠标悬浮头像时展示账户操作列表。
- 首页、导航、工具箱使用卡片网格和半透明磨砂面板。
- 门户导航和首页不展示后台管理入口。
- `/admin` 独立保留，用直接链接进入，并继续由超级管理员权限保护。

## 范围

本阶段包含：

- 全站品牌文案从“灵犀门户 / Lingxi Portal”调整为 `HLOVET`。
- 新增客户端顶部导航组件，用于读取登录状态、显示头像、等级徽章和账号菜单。
- 改造全局布局，从侧边栏工作台改为顶部导航 + 玻璃门户背景。
- 首页、导航、工具箱、登录、注册、工作台改为浅色磨砂卡片风格。
- 从公开门户导航和普通入口中移除管理后台入口。
- 管理后台保留，但只作为直接访问页面。

本阶段不包含：

- 新增真实头像上传。
- 新增后台入口权限配置。
- 新增深色模式。
- 新增导航/工具数据库 CRUD。
- 修改认证、角色或后台 API。

## 交互规则

- 卡片 hover 时轻微上浮，边框和阴影增强。
- 顶部导航使用磨砂半透明背景。
- 等级徽章使用 `title` 和自定义 tooltip 显示角色名与等级。
- 头像菜单包含工作台、导航、工具箱、退出登录。
- 超级管理员仍可以手动访问 `/admin`，但普通门户界面不提供该入口。

## 验收标准

- 页面可见品牌统一为 `HLOVET`。
- 顶部右侧未登录显示“登录”。
- 登录后显示等级徽章和用户头像。
- 悬浮头像显示账号操作列表。
- 门户导航和首页不出现管理后台入口。
- `/admin` 直接访问仍可进入权限判断和用户管理。
- `pnpm --filter @lingxi/web lint` 通过。
- `pnpm --filter @lingxi/web build` 通过。
- `pnpm build` 通过。
