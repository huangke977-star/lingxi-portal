# HLOVET Homepage 风格刷新设计

日期：2026-07-10

## 目标

参考 `gethomepage.dev` 的视觉组织方式，将 HLOVET 从“居中玻璃卡片门户”调整为更宽、更轻、更像个人入口面板的页面。顶部品牌和菜单不再放入悬浮框，主内容保留磨砂质感，但背后加入真实背景图，让玻璃层次更明显。

## 视觉方向

视觉主题：背景图、浅色磨砂、低阴影、宽版网格、轻量入口模块。

内容规划：

- 顶部品牌、菜单和账号区位于普通页面顶部，不使用悬浮框包裹。
- 主内容最大宽度扩大到接近桌面宽屏，避免页面挤在中间。
- 全局背景使用照片质感图片叠加柔和遮罩，卡片区域通过半透明和 blur 形成磨砂。
- 首页、导航、工具箱和工作台模块 hover 时只调整背景和边框，不再明显凸出或越过边界。
- 登录、注册和后台密码弹窗使用居中简洁面板，背景添加虚化遮罩，不放额外营销说明。
- `/admin` 继续只支持直接链接访问，不在门户菜单中出现。

## 移动端规则

- 默认显示首页内容。
- 顶部左侧显示菜单图标和 HLOVET 品牌标志。
- 顶部右侧显示登录入口或用户头像。
- 首页之外的导航菜单默认隐藏，点击左侧菜单图标后展开。
- 菜单项点击后自动收起。

## 范围

本阶段包含：

- 更新 `TopNav`，增加移动端菜单开关，并移除顶部导航的悬浮框视觉。
- 更新登录和注册页面，移除侧边说明文案，改为简洁的弹窗式表单。
- 更新全局 CSS，加入背景图层、宽版布局、轻量模块 hover 和统一弹窗遮罩。
- 保留已有认证、角色、头像、等级徽章和退出登录逻辑。

本阶段不包含：

- 修改后端 API。
- 新增真实头像上传。
- 新增数据库驱动的导航配置。
- 新增后台管理入口。

## 验收标准

- 顶部品牌和菜单不在悬浮玻璃框内。
- 桌面端内容区域明显比上一版更宽。
- 页面背后有背景图，磨砂面板有可见层次。
- 登录和注册页面没有多余说明区，面板居中且背景虚化。
- 功能模块 hover 不产生明显上浮或越界。
- 移动端默认隐藏菜单，通过左侧图标展开，右侧保留账号入口。
- `pnpm --filter @lingxi/web lint` 通过。
- `pnpm --filter @lingxi/web build` 通过。
